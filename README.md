# react-mobile-boilerplate
#### 介绍
- 基于 react + redux + react-router + typescript + webpack 的移动端单页应用项目模板
- 功能包括：调试、构建、代码规范校验、打包等


#### 运行

```
# 克隆项目
git clone https://github.com/laijunlinfz/react-mobile-boilerplate.git

# 进入项目目录
cd react-mobile-boilerplate

# 安装依赖
npm install

# 启动服务
npm run dev
```
